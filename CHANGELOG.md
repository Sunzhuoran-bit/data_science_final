# Changelog

本文件记录 `data_science_final` 项目的主要版本更新。版本号保持 `0.x` 阶段，最新版本放在最上方。

## 0.13.0 - 2026-06-15

### Added

- 新增重复题合并规则文件 `data/merge_overrides.js`，由 `docs/duplicate_report.md` 和 `docs/similar_report.md` 生成。
- 新增合并摘要 `data/derived/merge_summary.json`，记录合并来源、保留组、合并组数量和最终题量。
- 新增 `scripts/build_merge_overrides.js`，用于从重复题报告自动生成合并规则。
- 在题目卡片中加入 `重点` 标签：题目来源包含 `作业反馈_期末题库.pdf` 或 `作业反馈3.pdf` 时自动标注。

### Changed

- 按报告意见合并重复题与实质重复题：原始 286 题合并为 213 题。
- 合并题只保留一道题，但在题源中保留全部 PDF 出处，例如同一题同时显示作业反馈与期末题库来源。
- `docs/answer_key.md` 改为基于合并后的题库生成，答案表题号与网页题号保持一致。
- `data/derived/full_question_bank.json` 改为导出合并后的完整题库。
- 离线网页 `data_science_final.html` 同步使用合并后的题库与重点标注。

### Fixed

- 修正合并后题目来源丢失的问题。
- 修正答案表与网页题库数量不一致的风险。

## 0.12.0 - 2026-06-14

### Added

- 补全所有题目的答案与解析，集中维护在 `data/answer_overrides.js`、`data/final_bank_questions.js` 和生成后的 `docs/answer_key.md` 中。
- 新增专门的答案总表 `docs/answer_key.md`，按网页题号列出题干、选项、参考答案、解析和原始来源。

### Changed

- 修复期末题库中仅保留题号但缺少题干的条目。
- 修正第 169 题、第 171 题的题干与选项。
- 修正第 225 题答案：朴素贝叶斯假设的是给定类别后的条件独立，不是绝对独立。

## 0.11.0 - 2026-06-14

### Added

- 引入填空题人工判分流程。
- 填空题提交后只显示标准答案和解析，由答题人选择 `我答对了` 或 `我答错了`。
- 左侧题号格支持保留上次作答颜色，答对和答错使用不同状态。

### Changed

- 统计区文案从 `本人次数`、`本人正确率` 简化为 `作答次数`、`正确率`。
- 某个练习板块全部完成后，下次重新进入会重置该板块作答颜色，便于重新训练。

## 0.10.0 - 2026-06-14

### Added

- 合并早期版本与 v2 页面，形成单一正式版本。
- 网页标题统一为 `data_science_final`。
- 主页面、练习页、错题本、收藏夹、预测卷、训练记录、来源页统一路由。

### Changed

- `错题回收` 改名为 `错题本`。
- 首页大卡片和入口按钮整体居中。
- 返回按钮统一回到上一级操作，而不是直接退出网页。

## 0.9.0 - 2026-06-14

### Added

- 增加移动端适配版本。
- 主界面支持更紧凑的移动端按钮布局。
- 练习流程支持从首页进入、选择知识点、进入题组、再返回上一级重新选择。

### Changed

- 删除顶部多余说明文字。
- 优化手机端主视觉、按钮尺寸和页面间切换体验。

## 0.8.0 - 2026-06-14

### Changed

- 重新设计网页主界面，使首页只保留六个主要入口。
- 题号导航由长 ID 改为数字编号，降低侧栏占用。
- 难度从英文改为中文：`简单`、`中等`、`困难`，并使用颜色区分。
- 练习页改为按知识点进入题组答题。

## 0.7.0 - 2026-06-14

### Added

- 新增双击即可使用的离线单文件网页 `data_science_final.html`。
- 无需启动 Python 服务也可以完成答题、收藏、错题和记录功能。

### Changed

- 将网页依赖内联到单个 HTML 文件，便于分享和本地打开。
- 明确统计数据仅保存在本机浏览器 `localStorage`，不提供全站同步统计。

## 0.6.0 - 2026-06-13

### Added

- 新增完整覆盖型学案与题库核对流程。
- 增加 `docs/study_guides/data_science_final_study_workbook.docx`，以讲义式结构整理知识点、例题和练习。
- 增加 PDF 题号覆盖审计，避免遗漏不能归类的题目。

### Changed

- 学案不再按 lecture 粗分，而是按知识点逻辑组织。
- 无法归类的题目保留在独立分类中，不删除。

## 0.5.0 - 2026-06-13

### Added

- 扩展 PDF 原题覆盖，新增更多作业反馈题目。
- 每道题保留原始 PDF、页码、题型和题号。
- 增加补充题库文件与全覆盖题库文件。

### Changed

- 题库结构拆分为基础题库、补充题库、全覆盖题库和期末题库，便于维护。

## 0.4.0 - 2026-06-13

### Added

- 增加错题本功能。
- 增加收藏夹功能。
- 增加训练记录与预测卷得分记录。
- 增加个人作答次数和正确率统计。

### Changed

- 移除全站准确率与全站作答次数，避免静态网页无法同步造成误解。
- 答案反馈区改为以 `参考答案：` 为核心，并使用底色区分反馈状态。

## 0.3.0 - 2026-06-13

### Added

- 新增期末预测综合试题测试。
- 预测卷从 PDF 原题中抽取，不自编题。
- 完整答完后给出总分、知识点诊断和专项复习方向。

### Changed

- 预测卷按图片重点和期末题型要求打散考点。

## 0.2.0 - 2026-06-13

### Added

- 新增网页端交互答题系统。
- 支持按知识点选择题组。
- 支持判断题、单选题、多选题、填空题和开放题。
- 选择题答错后显示详细解析。
- 大题、填空题、开放式问答提交后显示参考答案。

### Changed

- 题目必须绑定 PDF 原始出处，不允许使用网络题源或自编题。

## 0.1.0 - 2026-06-13

### Added

- 初始化 `data_science_final` 项目。
- 建立开源项目式目录结构：`data/`、`docs/`、`src/`、`scripts/` 等。
- 新增 README，说明项目用途、运行方式、数据政策和目录结构。
- 建立初始数据科学期末复习题库与知识点分类。

### Notes

- 项目定位为本地静态复习工具。
- 所有题干、题目和答案来源必须可追溯到本地 PDF 或用户提供资料。
